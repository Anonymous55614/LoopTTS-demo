LoopTTS: Diagnose, Then Refine

A Closed-Loop TTS System with AudioLLM-Guided Correction

📄 Paper 💻 Code

Abstract

Current TTS systems adopt an open-loop, single-pass generation paradigm, inevitably producing sporadic local prosodic defects that conventional utterance-level metrics cannot detect. We present LoopTTS, a TTS system that places an AudioLLM Judge at its core and orchestrates two functionally complementary TTS models in a closed-loop Filter–Judge–Refiner architecture. A base TTS model produces an initial utterance; the Judge localizes word-level prosodic anomalies and generates structured refine instructions; a Refiner executes the instructions conditioned on the initial utterance. Experiments show that LoopTTS substantially improves prosodic naturalness through multi-round iterative refinement, bringing test-time compute scaling to TTS for the first time.
